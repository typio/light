This dataset contains the input variables (features) Sepal Length, Sepal Width, Petal Length, and Petal Width of 150 different Iris flowers belonging to the species (classes) Setosa, Versicolor, and Virginica.

This webpage trains a neural network, specifically a linear classifier, within your web browser y = Wx + b. x is a row of features in the dataset e.g. [5.1,3.5,1.4,0.2] and y is the predicted class e.g. Setosa. W is a matrix of weights which are multiplied by the input vector x, b is a vector which is then added to this product, the index of the largest value in the resultant vector is the classifiers predicited species.

W and b start off with random values. The neural network learns by adjusting W and b in the direction that will most minimize future error. Error is calculated with the softmax and cross-entropy functions.
